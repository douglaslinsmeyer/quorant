Elevation

Seven shadow levels from near-flat to modal depth. Light mode uses warm-tinted shadows; dark mode uses deeper black shadows for contrast.

--shadow-xs
Subtle borders, input fields
--shadow-sm
Cards, panels
--shadow-md
Dropdowns, popovers
--shadow-lg
Floating panels
--shadow-xl
Drawers, sheets
--shadow-2xl
Modals, dialogs
--shadow-inner
Inset inputs, wells

Motion

Purposeful animation reinforces spatial relationships and state changes. All animations respect prefers-reduced-motion.

Easing Curves

--ease-default cubic-bezier(0.25, 0.1, 0.25, 1)
--ease-out cubic-bezier(0.0, 0.0, 0.2, 1) — entrances
--ease-in cubic-bezier(0.4, 0.0, 1, 1) — exits
--ease-spring cubic-bezier(0.34, 1.56, 0.64, 1) — bouncy
--ease-smooth cubic-bezier(0.4, 0.0, 0.2, 1) — interactive
