Dark Mode Behavior

Toggle dark mode using the button in the top-right corner to observe how alert backgrounds adapt. The semantic-100 tokens invert to deep tinted surfaces in dark mode.

Dark mode token mapping: In dark mode, --color-success-100 resolves to #1A3D2A, --color-error-100 to #3D1A1A, --color-warning-100 to #3D2E0A, and --color-info-100 to #0E4F65. These deep, tinted surfaces maintain legibility while matching the dark UI palette. No additional overrides are needed in component code — the semantic tokens handle the adaptation automatically.
